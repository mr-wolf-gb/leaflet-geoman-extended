This project is based on [Leaflet-Geoman](https://github.com/geoman-io/leaflet-geoman) by [Geoman.io](https://geoman.io).

All core functionality and features are inherited from the original project. This extended version includes additional improvements and bug fixes while maintaining full API compatibility.

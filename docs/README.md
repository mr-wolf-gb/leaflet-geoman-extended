# Leaflet-Geoman-Extended Documentation

Welcome to the comprehensive documentation for Leaflet-Geoman-Extended, an extended version of the complete geometry editing plugin for Leaflet maps.

> **Note:** This is an extended version of the original [Leaflet-Geoman](https://github.com/geoman-io/leaflet-geoman) project by [Geoman.io](https://geoman.io). All core functionality and API remain compatible with the original.

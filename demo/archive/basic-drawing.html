Basic Drawing Tools

These are the fundamental drawing tools available in Leaflet-Geoman. Click any tool in the toolbar and then click on the map to start drawing. Most shapes require multiple clicks to define their boundaries.

How to Use

  1. Click a drawing tool in the toolbar
  2. Click on the map to start drawing
  3. For polygons/lines: click to add points
  4. Double-click or press Enter to finish
  5. Use edit mode to modify shapes
  6. Right-click for context menu options

Editing Features

After creating shapes, you can edit them using the toolbar buttons: drag to move, edit to modify vertices, rotate to change orientation, and scale to resize. The removal tool lets you delete selected shapes.

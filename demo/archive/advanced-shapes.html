Advanced Shape Tools

These specialized tools extend beyond basic shapes to provide advanced drawing capabilities including arrows, text labels, and cutting tools for complex geometric operations.

Advanced Tools Guide

  • Arrow: Click start point, then end point
  • Text: Click to place, enter text in modal
  • Cut: Draw line across shapes to split them
  • Freehand: Click and drag to draw smoothly
  • 3-Point Circle: Click three points on circumference

Pro Tips

All advanced shapes support the same editing features as basic shapes: drag to move, edit vertices, rotate, scale, and style customization. Use the context menu (right-click) for quick access to common operations.
